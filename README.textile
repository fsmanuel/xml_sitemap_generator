h2. XML Sitemap Generator

This is an approach to get the amazing XML Sitemap Generator for Wordpress to the rails world.

h3. Install

@script/plugin install git://github.com/fsmanuel/XML-Sitemap-Generator.git@

h3. Usage

Modify config/sitemap_config.yml to your needs

Include the MixinSitemap module in your model

@include MixinSitemap@
